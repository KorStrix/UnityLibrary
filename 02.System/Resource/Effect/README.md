### ㄴ 이미지를 클릭하시면 유튜브에서 비디오를 시청하실 수 있습니다.




- 이펙트 시스템은 사운드 시스템과 비슷한 설계입니다. 매니져와, 리소스 래퍼와, 플레이어로 이루어져 있습니다.

- Manager 주요 기능
  - Effect 풀링
  - 현재 몇개의 Effect가 플레이 중이고, 몇개를 생성했는지 지원
---
- CEffect 주요 기능
  - NGUI, Spine, Particle System을 지원.
  - 이펙트가 끝날 때 이벤트 통보를 지원.
